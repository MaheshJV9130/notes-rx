## 🎓 Learning Resources

### Understanding Next.js
- Built with Next.js 16 (App Router)
- See NOTES_RX_SYSTEM.md for tech stack
- All routes use JavaScript (no TypeScript)

### Understanding MongoDB
- Used Mongoose for ORM
- See SYSTEM_DOCUMENTATION.md - Database Schema
- All models include timestamps and references

### Understanding React
- Components are functional with hooks
- See code in `app/` directory
- Uses React 19.2

### Understanding APIs
- RESTful design with Next.js routes
- See API_REFERENCE.md for all 13 endpoints
- Examples in cURL and JavaScript
