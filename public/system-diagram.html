API Endpoints (13 Total)

Authentication

POST /api/auth/register - Create admin
POST /api/auth/login - Admin login
POST /api/auth/logout - Admin logout

Subjects

GET /api/subjects - All subjects
GET /api/subjects/[year] - By year
POST /api/subjects - Create subject

Chapters & Notes

GET /api/chapters?subject=[id]
POST /api/chapters - Create chapter
GET /api/notes?filters - Get notes
POST /api/notes - Create note

File Operations

POST /api/upload - Upload PDF
GET /api/pdf/[filename] - Stream PDF

2. Configure:

3. Run:
npm run dev
4. Access:
http://localhost:3000 - Home
http://localhost:3000/notes - Browse Notes
http://localhost:3000/admin-register - Create Admin
